# Welcome to CorpLink

CorpLink is a full-stack web application that streamlines **employee management, project tracking, performance reviews, and organizational communication**.

## What's Inside

CorpLink ships with a feature-rich set of modules:

- 📊 **Dashboard** with analytics and announcements
- 👥 **Employee Directory** and profiles
- 🌳 **Org Chart** visualization
- 🏖️ **Leave Management** with approval workflow
- ⏱️ **Timesheets** with submission and approval
- 📈 **Performance Reviews** (manager-restricted)
- 📁 **Projects**, **Assets**, **Policies**, **Certifications**
- 📋 **Surveys**, **Holidays**, **Expenses**, **Rewards**

## Tech Stack at a Glance

| Layer | Stack |
|-------|-------|
| **Frontend** | React 18 · Vite · Tailwind CSS · React Router · Axios |
| **Backend** | Node.js · Express · JWT · bcryptjs · Winston |
| **Database** | MySQL 8.0+ · 19 tables · seeded sample data |
