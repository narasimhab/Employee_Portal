## What's inside

CorpLink ships with a feature-rich set of modules:

- **Dashboard** with analytics and announcements
- **Employee Directory** and profiles
- **Org Chart** visualization
- **Leave Management** with approval workflow
- **Timesheets** with submission and approval
- **Performance Reviews** (manager-restricted)
- **Projects**, **Assets**, **Policies**, **Certifications**
- **Surveys**, **Holidays**, **Expenses**, **Rewards**
